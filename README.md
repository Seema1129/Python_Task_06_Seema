# Python_Task_06_Seema
# Student Cyber Security Management System

**Date:** 21 June 2026  
**Intern:** Seema

## Project Objective

The objective of this project is to combine all the programming concepts learned during the internship and develop a menu-driven application for managing student records and performing basic cyber security assessments. The project demonstrates the use of variables, input/output, conditional statements, loops, functions, lists, and file handling.

## Features

### Student Management
- Add Student
- View Students
- Search Student by Name or ID
- Delete Student Record

### Security Assessment
- Check Multi-Factor Authentication (MFA)
- Verify Password Length
- Check System Update Status
- Check Antivirus Installation
- Generate Security Score
- Display Security Status

### Report Generation
- Total Number of Students
- Security Scores
- Average Security Score
- Poor Security Ratings

### Cyber Security Features
- Password Strength Checker
- Username Generator
- Login Validation

### File Handling
- Save Student Records
- Read Student Records
- Store Data in `students.txt`

## Technologies Used

- Python
- Lists
- Functions
- Loops
- Conditional Statements
- File Handling
- Basic Cyber Security Concepts

## Program Flow

1. Display Main Menu.
2. Add Student Information.
3. View Student Records.
4. Search Student by Name or ID.
5. Delete Student Record.
6. Perform Security Assessment.
7. Generate Summary Report.
8. Check Password Strength.
9. Generate Username.
10. Validate Login Credentials.
11. Exit Program.

## Challenges Faced

- Managing multiple functions in a single application.
- Implementing file handling for storing records.
- Designing a menu-driven interface.
- Generating security scores.
- Handling invalid user input.
